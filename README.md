# Customer Churn Prediction

## Overview

This project predicts customer churn using Machine Learning techniques on the Telco Customer Churn dataset.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib
- Seaborn

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. One-Hot Encoding
5. Train-Test Split
6. Feature Scaling
7. Logistic Regression
8. XGBoost
9. Cross Validation
10. Model Evaluation

## Results

### Logistic Regression

- Accuracy: 78.75%
- Precision: 62%
- Recall: 52%
- F1 Score: 56%

### XGBoost

- Accuracy: 79.96%
- Precision: 66%
- Recall: 52%
- F1 Score: 58%

### Cross Validation

- Mean F1 Score: 60.65%
